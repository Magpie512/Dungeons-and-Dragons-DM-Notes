#### Rules, References & Databases

- **[D&D Beyond](https://www.dndbeyond.com)**

- **[5e.tools](https://5e.tools/)**

- **[Roll20 Compendium](https://roll20.net/compendium)**
	Free SRD rules, spells, and monsters.
- **[Wikidot 5e Homebrew](https://dnd5e.wikidot.com/)**

- **[GMBinder](https://www.gmbinder.com/)**
