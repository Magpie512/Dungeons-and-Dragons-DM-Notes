Here's a collection of my dungeon master notes for DnD 5e
Official Modules and otherwise. If I have ran it I have the notes.

#### Rules, References & Databases

- **[D&D Beyond](https://www.dndbeyond.com)**
	Character sheets, compendium, encounter builder, homebrew tools.
- **[5e.tools](https://5e.tools/)**
	Fast SRD lookup, monsters, spells, items, subclasses, encounter builder.
- **[Roll20 Compendium](https://roll20.net/compendium)**
	Free SRD rules, spells, and monsters.
- **[Wikidot 5e Homebrew](https://dnd5e.wikidot.com/)**
	Massive archive of community-created subclasses, races, items.
- **[GMBinder](https://www.gmbinder.com/)**
	Create polished, book‑style PDFs for homebrew content.
- **[The Homebrewery](https://homebrewery.naturalcrit.com/)**
	Turns markdown into D&D styled documents.

#### Maps, Tokens & Visuals
- **[Owlbear Rodeo](https://www.owlbear.rodeo/)**
	Lightweight VTT for quick sessions.
- **[Dungeon Scrawl](https://app.dungeonscrawl.com/)**
	Fast, browser-based dungeon map maker.
- **[Dungeon Map Doodler](https://dungeonmapdoodler.com/)**
	Simple drag‑and‑drop map creation.
- **[2-Minute Tabletop](https://2minutetabletop.com/)**
	Tokens, maps, assets, and art packs.
- [**Forgotten Adventures**](https://www.forgotten-adventures.net/)
	Huge library of map assets and tokens.

#### Generators & Randomizers

- **[Donjon](https://donjon.bin.sh/)**

- **[Kassoon](https://www.kassoon.com/)**

	Names for races, towns, ships, monsters, etc.
